# kindle-Land-of-Lisp

Land of Lisp is an awesome book.
I bought it from O'Reilly, but its figure is too small to read.
I tried to improve this issue.

## What you need

1. bundle gem
2. kindlegen

## What you do

1. bundle install
2. rake
3. transfer kindle/Land_of_Lisp.mobi to your kindle
4. read it!
